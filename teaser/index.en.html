PyCon Korea 2015

June 27–28

> in Korean

Call For Proposals

Overview

PyCon is a non-commercial conference held by Python Programming Communities over the world.

Since its inaugural edition in 2014, PyCon Korea has been running by a group of volunteers called ‘PyCon Korea Preparation Organizers’, who support the local Korean Python community. Please help us make PyCon Korea 2015 a conference with a full variety of sessions, a great place to share Python knowledge, and a fantastic networking opportunity.

  • Date : June 27th(Sat) – 28th(Sun), 2015
  • Venue : Nuri Dream Square, Sangam-dong, Seoul
  • Attendees: Appr. 700 people
  • Talk sessions : 50 mins long for each sessions (can be changed)

Topics

We accept a broad range of Python-related proposals, from academic reports to commercial projects and case studies. Suitable topics include, but not limited to:

  • Python Projects

    • Web Programming (e.g. Django, Flask, Pyramid)
    • Cloud Computing / Network Programming
    • System Administration and Security
    • Multimedia, Animation and Computer Graphics
    • Game / GUI Programming
    • Hardware / Embedded System Design
    • SciPy and Numpy
    • Big Data / PyData, Data Mining, Machine Learning
    • Bioinformatics
    • Deploying Applications

  • Python Language

    • Packaging Issues
    • Programming Tools
    • Python Libraries and Extensions
    • Python Implementations: IronPython, Jython, PyPy and Stackless

  • Python and Business

    • Agile Development and Project Best Practices
    • Startups, Business, and Education

  • Python for Beginners
